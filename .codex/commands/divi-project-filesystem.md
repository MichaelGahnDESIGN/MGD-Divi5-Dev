# /divi-project-filesystem

Lege die Projektordner-Struktur mit MGD-AI-Basic-Projektordner an.

## Verhalten

Nutze `skill/DIVI5-SKILL.md` als Arbeitsgrundlage und führe den Abschnitt `/divi-project-filesystem` durch:

1. MGD-AI-Basic-Projektordner als Basis verwenden (https://github.com/MichaelGahnDESIGN/MGD-AI-Basic-Projektordner)
2. Projektspezifische Ordnerstruktur anlegen
3. `.gitignore` für WordPress anpassen
4. `docker-compose.yml` hinzufügen

## Regeln

- Neues Projekt IMMER auf Basis von MGD-AI-Basic-Projektordner starten
- `.gitignore` muss wp-config.php, .env, backups/, uploads/ ausschließen
- Sinnvolle README.md für das Projekt anlegen

## Ergebnis

Am Ende kurz berichten:
- Welche Ordnerstruktur angelegt wurde
- Welche nächsten Schritte empfohlen werden (/divi-install oder /divi-install-docker)
